- Expressjs(API): (Done)Expressjs + ryou-router
- KOA(API): (Pending)
- Nginx spa history configuration: (Done)

## Use

Install：
```sh
  npm install -g ryou-cli
```

Run：
```sh
  ryoucli init <ProjectName>
```
